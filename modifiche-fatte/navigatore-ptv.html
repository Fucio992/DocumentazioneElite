🛠️ Istruzioni Tecniche per Installazione PTV Navigator

Guida sintetica per tecnici incaricati dell’attivazione e integrazione del modulo PTV Navigator con Elite.

📦 Prerequisiti

  • Dispositivo Android aggiornato
  • App e-TIR installata e aggiornata dal Play Store
  • File APK disponibile per installazione manuale:
  • Scarica PTV_NavigatorG2.apk

🔐 Attivazione Licenza

  1. Installare l'app APK sul dispositivo del cliente
  2. Scansionare il QR Code fornito via e-mail (contenente License Key)
  3. Verificare che la chiave sia attiva e corretta

🧩 Attivazione nel software Elite

  1. Aprire la configurazione del software
  2. Verificare aggiornamento menu e API
  3. Abilitare il servizio PTV Navigator
  4. Salvare la configurazione

Questa operazione abilita il salvataggio del percorso (BCR) nella tabella ElencoViaggiTestataPercorsi.

📅 Appuntamento con assistenza

Un appuntamento con l’assistenza sarà già fissato per completare la configurazione e offrire supporto formativo.

Nota: se non ancora ricevuto, chiedere conferma dell’orario all’ufficio assistenza.
