    • 👨🏻‍💻 Comandi per compilare i programmi flutter:

    Ci sono alcuni comandi specifici per la compilazione dei programmi flutter:
    • Creare le funzioni FROM e TO JSON automatiche utilizzando il pacchetto build 
      dart run build_runner build --delete-conflicting-outputs
    • Per compilare l'app in Android 
      flutter build appbundle
    • Per compilare l'app in iOS 
      flutter build ios
      Poi andate su XCode e premete Archive e aspettate che compili
    • Per compilare i portali in web 
      flutter build web --no-tree-shake-icons
    • Per compilare l'app in Windows 
      flutter build windows
      e poi cercate la cartella da condividere su questo percorso: 
      {PERCORSO DEL PROGETTO}/build/windows/x64/runner\Release\app_horizon_tir.exe

    📜 Regole generali dei portali:

    I portali, a differenza delle applicazioni, si collegano alle API tramite il file config.json che appare così: 
    
            {
              "cliente": "NomeCliente",
              "pi": "00011122233"
            }
    dove l'abbinamento viene fatto tramite PI con una chiamata al server dell'Assistenza dove sono registrati tutti i link delle API di ogni cliente. Questo vuol dire che per accedere al portale di qualcun altro, basta inserire la sua partita iva nel config.
    Inoltre, essendo che quando si installa l'API viene sempre creato l'utente assistenza, potete accedere ad ogni portale con le seguenti credenziali:
    Usr: assistenza@ideagrip.it
    Psw: Ide@Grip-1
